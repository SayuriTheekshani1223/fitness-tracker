# Fitness Tracker - BalanceBeat

A web-based fitness tracking application built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features

- User registration and login
- Activity logging with duration and calories
- Contact form
- Dashboard for users
- Responsive design

## Setup Instructions

### Prerequisites
- XAMPP server
- PHP 7.4 or higher
- MySQL

### File Structure


Fitness-tracker/
├── includes/
│   ├── db.php          # Database connection
│   └── functions.php   # Helper functions
├── auth/
│   ├── register.php    # User registration
│   ├── login.php       # User login
│   └── logout.php      # Logout
├── contact.php         # Contact form
├── dashboard.php       # User dashboard
├── Activity.php        # Activity logging
├── Goals.php           # Goals setting and tracking
├── Progress.php        # Progress visualization
├── Home.php            # Home page
├── database.sql        # Database schema
├── style.css           # Stylesheet
└── README.md           # This file


### Usage

1. Register a new account or login
2. Log activities in the Activity page
3. View your dashboard
4. Contact via the contact form

## Technologies Used

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL
